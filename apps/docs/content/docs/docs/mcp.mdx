Kibo UI supports the Model Context Protocol (MCP) for model-driven development.

## Installation

Copy and paste the code into `.cursor/mcp.json` or `.codeium/windsurf/mcp_config.json`:

```json
{
  "mcpServers": {
    "shadcn": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.kibo-ui.com/api/mcp/http"
      ]
    }
  }
}
```
